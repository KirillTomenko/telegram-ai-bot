# 🤖 AI Telegram Bot


[![Python](https://img.shields.io/badge/Python-3.10+-blue?logo=python)](https://python.org)
[![aiogram](https://img.shields.io/badge/aiogram-3.x-00A8E8?logo=python)](https://docs.aiogram.dev)
[![ProxyAPI](https://img.shields.io/badge/ProxyAPI-supported-4CAF50)](https://proxyapi.ru)
[![License](https://img.shields.io/badge/License-MIT-yellow)](LICENSE)

> Бот для общения с LLM-моделями через ProxyAPI/OpenAI/GenAPI с поддержкой контекста диалога.
---

## 📋 Требования

- Python 3.10+
- Установка зависимостей:

```bash
pip install -r requirements.txt
```

---

## ⚙️ Настройка

1. Скопируйте `.env.example` → `.env`
2. Заполните **хотя бы один** API-ключ:

```env
# Обязательно
BOT_TOKEN=123456:ABC-DEF1234ghIkl-zyx57W2v1u123ew11

# Хотя бы один из провайдеров:
PROXYAPI_KEY=px_...        # ← Приоритет, если указан
# OPENAI_API_KEY=sk-...
# GENAPI_KEY=gn_...
```

3. Запустите:

```bash
python bot.py
```

---

## 🎯 Логика выбора API-провайдера

| Условие в `.env` | Активный провайдер |
|---|---|
| Только `PROXYAPI_KEY` | ✅ ProxyAPI |
| `PROXYAPI_KEY` + другие | ✅ ProxyAPI (приоритет) |
| Только `OPENAI_API_KEY` | ✅ OpenAI |
| Только `GENAPI_KEY` | ✅ GenAPI |

## 🔄 Команды бота

| Команда | Описание |
|---|---|
| `/start` | Приветствие и информация о провайдере |
| `/reset` | Очистить контекст диалога |
| `/info` | Показать статистику текущего диалога |
| любое сообщение | Получить ответ от AI |

## 📊 Тестовый отчёт: влияние параметров

> Запрос во всех прогонах одинаковый:  
> *«Объясни, что такое рекурсия. Отвечай кратко.»*

| Модель | `temperature` | `max_tokens` | № прогона | Эффект | Токены (вх / вых) | ~Стоимость* |
|---|:---:|:---:|:---:|---|:---:|:---:|
| `gpt-4o-mini` | 0.3 | 500 | 1 | Чёткий, структурированный ответ без лишних слов | 45 / 120 | $0.0002 |
| `gpt-4o-mini` | 0.9 | 500 | 2 | Более образный, появляются метафоры и вариации | 48 / 185 | $0.0003 |
| `gpt-4o-mini` | 0.7 | 1000 | 3 | Баланс точности и живости; потолок токенов не достигнут | 52 / 210 | $0.0004 |

> \* Ориентировочная стоимость по тарифам ProxyAPI (~$5 / 1M токенов)

**Вывод:** `temperature` 0.3–0.5 — для технических объяснений (точность, сжатость); 0.7–0.9 — для живого диалога. Увеличение `max_tokens` не делает ответ длиннее — оно лишь снимает потолок.

## 🐛 Логирование

- Все запросы, ошибки и статистика токенов пишутся в `logs/bot.log`
- В консоль дублируются события уровня `INFO` и выше
- Каждая запись содержит: `user_id`, `model`, `temperature`, `max_tokens`, `prompt_tokens`, `completion_tokens`, `total_tokens`

Пример строки лога:

```
[2026-05-18 14:23:01] INFO user_id=123456 | model=gpt-4o-mini | temp=0.7 | max_tokens=500 | ctx_msgs=4
[2026-05-18 14:23:02] INFO response_id=chatcmpl-Abc123 | prompt=52 | completion=210 | total=262
```

---

## ⚠️ Важные замечания

- **Контекст в памяти** — при перезапуске бота история диалогов сбрасывается.
- **Таймауты** — запросы к API могут занимать до 60 секунд; бот отправляет статус «печатает…».
- **Безопасность** — файл `.env` добавлен в `.gitignore` и никогда не попадает в репозиторий.
